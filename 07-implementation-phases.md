- Validate PowerToys interest.
- Confirm that on-demand temporary mode is the preferred direction.
- Lock V1 scope to desktop folders only.

- Detect open Explorer folders.
- Build desktop item index.
- Resolve desktop icon coordinates.
- Prove temporary renderer concept.
